# Хроники войны

Исторический пошаговый тактический симулятор битв на гексах.

## v0.1 MVP Bootstrap

Сейчас есть первая рабочая версия без картинок:

- главное меню;
- выбор битвы;
- выбор стороны;
- пошаговый бой на гексах;
- движение;
- атака;
- простая мораль;
- простая победа/поражение;
- боевой журнал;
- результат боя;
- чистое разделение `core / data / content / features / ui`.

## Команды

```bash
npm install
npm run dev
npm run build
```

## Архитектурное правило

UI показывает состояние и отправляет команды. Боевые правила живут в `src/core`.
Исторические данные и сценарии живут в `src/data` и `src/content`.
